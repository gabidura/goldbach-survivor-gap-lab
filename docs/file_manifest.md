# File manifest

## Source

- `src/goldbach_lab_v03.cpp`  
  C++17 source for Goldbach Lab v03.

## Scripts

- `scripts/build.sh`  
  Builds the C++ executable.

- `scripts/run_first_interior_10B.sh`  
  Runs the exhaustive first-interior scan up to `10^10`.

- `scripts/run_gap_sample_10B.sh`  
  Runs a sampled complete-window gap scan up to `10^10`.

- `scripts/run_exact_gap_1M.sh`  
  Runs an exact complete-window gap scan up to `10^6`.

- `scripts/run_single_example.sh`  
  Runs a single large diagnostic with the Miller backend.
